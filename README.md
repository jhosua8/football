# FootballHub
Proyectode GIT - Footballhub la realizamos para la asignatura de Mòdulo professional optatiu.

Consiste en una aplicación web que muestra información sobre equipos, tabla de clasificación, partidos y formulario de contacto.

## Tecnologías utilizadas
- HTML
- CSS
- JavaScript
- JSON
- Git Bash
- GitHub

## Estructura del proyecto
- `/pages` → páginas HTML (equipos, tabla, partidos, contacto)
- `/js` → scripts de renderizado, filtros y validaciones
- `/data` → archivos JSON con datos de equipos, tabla y partidos
- `/css` → estilos del proyecto

## Funcionalidades
- Listado de equipos con filtro por país  
- Tabla de clasificación con ordenación por columnas  
- Partidos con filtro por equipo  
- Formulario de contacto con validación  
- Gestión de ramas, merges y resolución de conflictos con Git
